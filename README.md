# React - Furniture Project

## Stack

- React
- React Hooks
- Json-server
- Axios
- SCSS / CSS-modules
- React Content Loader (React Skeleton)
- React Router

### Realised

- Home Page
- Orders Page
- Cart Page
- WishList Page
- Pagination
- Routing
- Slider
- Preloading Items with React Skeleton
- Catalog Item Search
